Furns Setup (React)

Node.js and Yarn

Your computer must have installed nodejs and yarn to run this template. You can download Node.js from https://nodejs.org and yarn from https://yarnpkg.com/lang/en/docs/install/. NPM comes bundled with Node.js







Install Npm Packages

You must have to install packages. You can do this by running yarn install from the root of your project to install all the necessary dependencies.




Development server

Run yarn dev from the root of your project to start a dev server. Navigate to http://localhost:3000/. The app will automatically reload if you change any of the source files.




Build

Run yarn build to build the project. The build artifacts will be stored in the .next/ directory.




Export

Run yarn export to build the project. The build artifacts will be stored in the out/ directory.




Eject

Run yarn eject If you are not satisfied with the default configuration. Note: this is a one-way operation. Once you eject, you can’t go back!




Shopify Setup

Shopify's Storefront API allows you to build custom ecommerce experiences.

Requirement

  • STOREFRONT NAME
  • STOREFRONT ACCESS TOKEN

How to set up Shopify Partners account and create a new store:

Navigate to Shopify Partners and create your Shopify Partners account (if you don't have one already). Then, navigate to the Stores tab on the left-hand side and create a new store. Among other benefits, a Partners account will allow you to easily manage stores and have unlimited time to work on them before transferring to a client.

Connect Shopify store to Next.js app

Navigate to the Apps tab in your store and hit the "Manage private apps" link at the bottom

Accept the terms and conditions, name your private app "Next.js Connection" and enter your email address. Then scroll down and check "Allow this app to access your storefront data using the Storefront API".

Now, open .env.local file and edit these variables:

NEXT_PUBLIC_SHOPIFY_STOREFRONT_NAME='storefront name'
NEXT_PUBLIC_SHOPIFY_STOREFRONT_ACCESS_TOKEN='storefront api access token'

Get storefront name from domain. e.g. graphql.myshopify.com. Here, "graphql" is storefront name.

Step #2: Go to your Shopify Admin and click on the “Products” tab:

Step #3: Click “Add a product” located at the top right-hand corner:

Step #4: Enter your product’s “Title”, “Description” & Others info:

How to add collection

Step #1: View collection list

After you login in your Shopify account, choose Products. In this section, select Collections and all the collections that you have created will be displayed.


Step #2: Click Create collection

Step #3: Enter collection details:

Step #4: Select Manual as Collection type:

Step #5: Add product to collection:

When your collection has been created, the next thing you will have to do is add some products to your manual collection. Click Add products to view your product list. Then from that list, you can choose several products to your collection.



Step #6: Save new collection:

After that, click Save so that your new collection could be added.


More Details:
https://help.shopify.com/en/manual/products/collections/manual-shopify-collection
